## 💡 프로젝트 실행 방법
#### 1. 프로젝트 패키지 설치
```
npm install axios
npm install react-router-dom
npm i --save-dev @types/styled-components
npm i styled-components
npm install react-icons --save
```
#### 2. 프로젝트 실행
```
npm start
```
